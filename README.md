# Blog_app
Project Title
Blog Application

Description
This is a simple Blog application built using Django. It allows users to create blog posts, view posts, edit posts, and delete posts. The project is designed to help beginners learn how to work with Django models, views, templates, and CRUD operations.

Features
Create new blog posts
View all blog posts
View individual blog details
Edit existing blog posts
Delete blog posts
User friendly interface
Clean and simple layout

Technologies Used
Python
Django
HTML
CSS
SQLite database

How to Install and Run the Project

Install Python on your system

Open a terminal and go to the project folder

Create a virtual environment using the command
python -m venv venv

Activate the virtual environment
Windows: venv\Scripts\activate
Mac or Linux: source venv/bin/activate

Install Django
pip install django

Run the migrations
python manage.py migrate

Start the development server
python manage.py runserver

Open your browser and visit
http://127.0.0.1:8000/

Project Structure
blog app handles all blog related operations
models file defines the blog post structure
views file contains the logic for handling requests
templates folder contains the HTML pages for listing, showing, creating, and editing posts


Usage
Open the application in your browser
Create new blog posts
View and read blogs
Edit or delete blogs as needed
Use it to manage your personal or demo blog easily
